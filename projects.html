Memory Card Game (Jan 2021 - Feb 2021)

This project was a continuation on learning and getting more comfortable with the pygame library and python overall. It is a version of a memory card matching game. There are a certain amount of cards which are flipped over and each contain an image of a fruit. Each fruit is represented twice in the cards. It is up to the user to attempt to flip two cards at once and match them together. If the image is not the same then the cards flip over and the user trys again. The game completes once the user has flipped and correctly matched each pair in the deck. The user is then prompted with whether or not they would like to restart and try again.

Snake (Dec 2020 - Jan 2021)

I started this project to further my knowledge in python and the pygame library. It is a simple game recognizable as the well known Snake game. Where the user controls the head of the snake and attempts to eat food to make the snake grow larger. The user wins when the snake reaches a size designated by the game before it starts. If the user causes the snake to run into itself or off the screen then the game is over and the user has the option to reset and try again. Overall, this was a pretty quick project which allowed me to become more comfortable with python and programming using the pygame library.
